Múltiplas Instâncias por Tenant

Multi-Instance

Cada tenant pode ter múltiplas instâncias WhatsApp conectadas simultaneamente. A listagem carrega todas as instâncias via WhatsappInstance::orderBy('id')->get() e considera o sistema como conectado se qualquer instância estiver com status connected.

  • Cada conversa (WhatsappConversation) possui referência à instância via instance:id,label,phone_number
  • Instâncias podem ter labels distintos para identificação (ex: "Vendas", "Suporte")
  • allInstances.count() é passado à view para exibição de indicadores
  • Somente admin pode conectar ou desconectar instâncias

Filtragem por Departamento

Lógica de filtragem (WhatsappController): Se o usuário não é admin (!$authUser->isAdmin()), as conversas são filtradas para exibir apenas: (1) conversas cujo department_id pertence aos departamentos do usuário, (2) conversas sem departamento (department_id IS NULL), ou (3) conversas atribuídas diretamente ao usuário (assigned_user_id). 
// WhatsappController@index — filtragem por departamento
$restrictByDept = !$authUser->isAdmin();
$userDeptIds = $restrictByDept
    ? $authUser->departments()->pluck('departments.id')
    : collect();

if ($restrictByDept) {
    $waQuery->where(function ($q) use ($authUser, $userDeptIds) {
        $q->whereIn('department_id', $userDeptIds)
          ->orWhereNull('department_id')
          ->orWhere('assigned_user_id', $authUser->id);
    });
}

Automações Instagram

Auto-Reply por Keywords

O sistema de automação Instagram permite configurar respostas automáticas baseadas em palavras-chave de comentários em posts específicos.

  • Configuração: Admin seleciona um post, define keywords de trigger, mensagem de reply no comentário e/ou DM automática
  • Fluxo: Comentário recebido → match de keyword → reply público + envio de DM privada com conteúdo configurado
  • Gerenciamento: Somente admin pode criar, editar e excluir automações Instagram
  • Model: InstagramAutomation — armazena post_id, keywords, reply_text, dm_text

Filtragem por Departamento (Instagram)

Mesma lógica do WhatsApp: Conversas Instagram (InstagramConversation) também são filtradas por departamento para Manager e Viewer. A query aplica o mesmo padrão: whereIn('department_id', $userDeptIds)->orWhereNull('department_id')->orWhere('assigned_user_id', $authUser->id). Admin vê todas sem restrição.

Inbox Unificado

As conversas de WhatsApp e Instagram são unificadas no mesmo inbox, ordenadas por last_message_at desc. Cada conversa recebe um atributo _channel ('whatsapp' ou 'instagram') para identificação visual na UI.
