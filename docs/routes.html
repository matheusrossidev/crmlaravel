Rotas

Mapa completo de rotas web, API e channels

Sistema de permissoes por role: Rotas de escrita (POST/PUT/DELETE) sao protegidas pelo middleware role. Existem tres niveis de acesso:

admin — Apenas administradores. Usado em configuracoes (pipelines, tags, integracoes, automacoes, chatbot, IA, cobranca, API keys, departamentos, usuarios, campos extras, Instagram automations).
admin+manager — Administradores e gestores. Usado em operacoes CRM (leads CRUD, calendario, chat escrita, campanhas, importacao, movimentacao de etapas).
all roles — Qualquer usuario autenticado. Usado em rotas GET/leitura e perfil pessoal.

Rotas publicas (auth, webhooks, widget) nao possuem restricao de role.
