Banco de Dados

Schema, migrations, seeders e convenções

Convenções

defaultStringLength(191): Setado em AppServiceProvider. MySQL do WAMP tem max key length de 1000 bytes. Índices compostos com 3+ colunas varchar podem exceder esse limite.
ConvençãoMotivo
Schema::defaultStringLength(191)Compatibilidade com MySQL antigo (utf8mb4 × 4 bytes × 191 = 764 bytes)
contact_picture_url deve ser TEXTURLs do WhatsApp excedem 191 chars
phone deve ser VARCHAR(30)LID numéricos: até 14 dígitos; WhatsApp: até 15
Evitar índice composto com 3+ varcharUltrapassa max key 1000 bytes
JSON em colunas *_jsonCasts automáticos no Eloquent ('array')
$timestamps = false em vários modelsMuitas tabelas usam apenas created_at
